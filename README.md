# Excel Data Cleaning Project

## Project Overview
This project focuses on cleaning and preparing a raw dataset using Microsoft Excel.  
The objective is to convert unstructured and inconsistent data into a clean, analysis-ready format.

## Files Included
- raw_data.xlsx → Original dataset before cleaning
- cleaned_data.xlsx → Final cleaned dataset

## Data Cleaning Steps Performed
- Removed duplicate records
- Handled missing values
- Standardized column names
- Corrected data formats (dates, numbers, text)
- Removed unnecessary columns
- Checked data consistency and accuracy

## Tools Used
- Microsoft Excel
  - Filters
  - Find & Replace
  - Text to Columns
  - IF, TRIM, CLEAN functions
  - Conditional Formatting

## Outcome
The cleaned dataset is structured, accurate, and ready for further analysis or visualization.
